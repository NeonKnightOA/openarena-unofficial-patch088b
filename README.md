# Neon_Knight's unofficial OpenArena patch ver. 0.8.8b.

This is an asset-only patch aiming to fix errors on maps and game assets and possibly enhance them; no gamecode involved. For the gamecode, you go [here](https://github.com/OpenArena/engine) and [here](https://github.com/OpenArena/gamecode). (And, while we're at it, [here](https://github.com/OpenArena/UI3)). For downloadable gamecode vms, go [here](http://files.poulsander.com/~poul19/public_files/oa/dev088/gamecode_nightly/).

Also, if the title wasn't enough indication, this patch, despite its aim, is **non officially endorsed**. You shouldn't worry about dangers, though. It's all safe.

The changelog and roadmap for this patch is located in the Docs folder.
