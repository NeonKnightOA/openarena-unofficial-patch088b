== General ==
* Many shaders are now displayed correctly in the editor.

== Map specific ==
* In general
  * All fixed maps use the newer jumppad/item base SFX.
  * All maps have message, author and music listed.
  * All fixed maps were recompiled using NRC Q3 Final settings.
  * All fixed maps have new levelshots.
  * All instances of common/nodraw were replaced by common/nodrawnonsolid.
  * All instances of info_player_start were replaced by info_player_deathmatch with spawnflags "1" (Initial).
  * All maps include an .info file.
